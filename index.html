Crystal Collector

Game Rules and Instructions

Welcome to Crystal Collector. At the start of the game you will be given a random number. Your objective is, by clicking the crystals below, to match the value of the number given to you. If you go over the value of the random number then it will count as a loss but if you match the value of the random number you win! Each crystal is given a random value at the start of each round to keep the game fresh.
